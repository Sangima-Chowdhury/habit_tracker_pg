# PostgreSQL Habit Tracker

A Flask backend project connected to a PostgreSQL database. This project was built to practise using PostgreSQL with Flask and SQLAlchemy.

## Features
- Flask application setup
- PostgreSQL database connection
- SQLAlchemy model creation
- Database table creation
- Production-style database setup

## Technologies Used
- Python
- Flask
- SQLAlchemy
- PostgreSQL
- psycopg2-binary
- Git & GitHub

## What I Learned
This project helped me understand how Flask connects to PostgreSQL, how database models work, how primary keys identify records, and how production-style databases differ from SQLite.
